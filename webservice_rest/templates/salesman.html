API de Vendedores

provee acceso a los vendedores, se pueden crear editar y agregar

Changelog

21-04-2014

Primera versión

/salesman/add

Para editar un vendedor se debe utilizar /salesman/edit, incluyendo sólo los parámetros que se deseen editar.

Parámetros

name

 String

Nombre del vendedor

password

 String

Contraseña con la que se logeará el usuario

email

 String

Email del usuario, será usado para login

id

 String

Usar sólo en caso de querer editar

Ejemplo de salida

                                {"success": "53558dda4d682d24d8255618"}

/salesman/find

Parámetros

email

 String

Email del usuario a buscar

Ejemplo de salida

                            {
    "email": "a@a.com",
    "password": "test",
    "name": "test",
    "_id": {
        "$oid": "53558dda4d682d24d8255618"
    }
}

/category/remove

Parámetros

email

 String

email del usuario que se eliminará

Ejemplo de salida

                                {"success": "object: 53558dda4d682d24d8255618 has been deleted"}

/salesman/list

Parámetros

page

 Int

Número de página

items

 Int

Número de elementos que se mostrarán por página.

Ejemplo de salida

                            [
    {
        "email": "a@a.com",
        "password": "test",
        "name": "test",
        "_id": {
            "$oid": "53558dda4d682d24d8255618"
        }
    }
]
