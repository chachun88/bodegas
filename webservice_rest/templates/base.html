API de Producto

provee acceso a los productos, se pueden crear editar y agregar

Changelog

21-04-2014

Primera versión

/product/add

Para editar un producto se debe utilizar /product/edit, incluyendo sólo los parámetros que se deseen editar.

Parámetros

sku

 String

Código único del producto

name

 String

Nombre del producto

upc

 String

description

 String

Descripción larga del producto.

brand

 String

Marca del producto, si esta no existe, será creada automáticamente

manufacturer

 String

Nombre del fabricante, si no existe, este se crea automáticamente

color

 String

Color, o listado de colores separados por coma. Si no existe, se crea automáticamente

material

 String

Material o lista de materiales separados por coma, si no existe, este se crea automáticamente

bullet_1

 String

bullet_2

 String

bullet_3

 String

price

 Float

Pecio de compra del producto al momento de crearlo

image

 String

URL para imagen principal del producto

image_2

 String

URL para la segunda del producto

image_3

 String

URL para la tercera imagen del producto

Ejemplo de salida

                                {"success": "product correctly saved"}

/product/find

Parámetros

sku

 String

Encuentra un producto según el identificador indicado

Ejemplo de salida

                                {
    "sku": "1",
    "category": "",
    "image_3": "",
    "image_2": "",
    "name": "testing_product",
    "color": "",
    "brand": "giani",
    "material": "",
    "_id": {
        "$oid": "53557aa19ec9a724d812b587"
    },
    "manufacturer": "",
    "bullet_point_1": "",
    "description": "alguna",
    "bullet_point_3": "",
    "image": "",
    "bullet_point_2": "",
    "size": ""
}

/product/remove

Parámetros

sku

 String

Se borrarán todos los registros de productos con este sku.

Ejemplo de salida

                                {"success": "object: 53557aa19ec9a724d812b587 has been deleted"}
