API de Categorías

provee acceso a las categorías, se pueden crear editar y agregar

Changelog

21-04-2014

Primera versión

/category/add

Para editar una marca se debe utilizar /category/edit, incluyendo sólo los parámetros que se deseen editar.

Parámetros

name

 String

Nombre de la categoria

id

 String

Usar sólo en caso de querer editar el nombre de una categoria

Ejemplo de salida

                                {"success": "category saved correctly"}

/category/find

Parámetros

id

 String

Identificador de la categoría a encontrar

Ejemplo de salida

                            {"_id": {"$oid": "535585a34d682d24d801fc7f"}, "name": "categoria", "parent": ""}

/category/remove

Parámetros

id

 String

Identificador de la categoría

Ejemplo de salida

                                {"success": "object: 535585a34d682d24d801fc7f has been deleted"}

/category/list

Parámetros

page

 Int

Número de página

items

 Int

Número de elementos que se mostrarán por página.

Ejemplo de salida

                            [
    {
        "_id": {
            "$oid": "534fdfa99ec9a710fb16590f"
        },
        "name": "zapatos",
        "parent": ""
    },
    {
        "_id": {
            "$oid": "535585a34d682d24d801fc7f"
        },
        "name": "categoria",
        "parent": ""
    }
]
