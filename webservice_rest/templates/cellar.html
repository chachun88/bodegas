API de Bodegas

provee acceso a las bodegas, se pueden crear editar y agregar

Changelog

21-04-2014

Primera versión

/cellar/add

Para editar una bodega se debe utilizar /bodega/edit, incluyendo sólo los parámetros que se deseen editar.

Parámetros

name

 String

Nombre de la bodega

description

 String

Descripción detallada de la bodega

id

 String

Usar sólo en caso de querer editar el nombre de una bodega

Ejemplo de salida

                                {"success": "535580649ec9a724d88634ac"}

/cellar/find

Parámetros

id

 String

Identificador de la bodega a encontrar

Ejemplo de salida

                            {
                                "_id": {
                                    "$oid": "535580649ec9a724d88634ac"
                                },
                                "name": "testing_cellar",
                                "description": "this is a testing cellar"
                            }

/cellar/remove

Parámetros

id

 String

se borrará la bodega con todas las unidades de productos guardados en ella.

Ejemplo de salida

                                {"success": "object: 535580649ec9a724d88634ac has been deleted"}

/cellar/list

Parámetros

page

 Int

Número de página

items

 Int

Número de elementos que se mostrarán por página.

Ejemplo de salida

                            [
                                {
                                    "_id": {
                                        "$oid": "535580649ec9a724d88634ac"
                                    },
                                    "name": "testing_cellar",
                                    "description": "this is a testing cellar"
                                }
                            ]

/cellar/products/list

Parámetros

id

 String

Identificador de la bodega

page

 Int

numero de pagina para paginacion

items

 Int

Cantidad de elementos que se mostrarán por página

/cellar/products/add|remove|move

Parámetros

cellar_id

 String

Identificador de la bodega

product_id

 String

Identificador del producto

quantity

 Int

Cantidad de productos que se agregarán

operation

 String

Tipo de operación, puede ser "buy" o "sell"

price

 String

Precio de compra, sólo aplica para compras
